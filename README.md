# pegonpad
Editor Teks sederhana untuk mempermudah memasukkan aksara pegon saat mengetik.
Aplikasi hanya berupa 1 file kecil dalam format HTML, dibuka dengan browser.
Aplikasi ini cocok bagi anda yang telah lancar/bisa mengetik arab namun terkendala memasukkan karakter pegon.

Fitur Utama :
- Tombol-tombol aksara pegon untuk menambahkan huruf pegon, karena keyboard fisik tidak menyediakannya.
- Simbol shallallahu alaihi wasallam (ﷺ), hamzah washal, beberapa harakat khusus dan tanda baca penulisan mushaf. Hal ini sering diperlukan untuk layout kitab dzikir, maulid atau manaqib.
- Angka Arab Hindi (١٢٣) permanen, tersedia bagi anda bila angka arab yang anda ketik memunculkan angka arab barat (123).

Fitur Tampilan :
- Tema Terang/Gelap.
- Pilihan font yang ditampilkan pada aplikasi.
- Ukuran font pada tampilan aplikasi.
- Perbesar-perkecil tombol.

Karakter Pegon tidak didukung semua font Arab.
Font utama adalah Pegon Virtual. Silakan unduh dari https://nawaksara.id/wp-content/uploads/2025/01/Pegon-Virtual.ttf.
Diantara font lain yang mendukung font arab adalah;
- Amiri.
- ArabicTypeSetting.
